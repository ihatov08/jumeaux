Installation
============

First, Clone this repository.


Install requisite packages
--------------------------

.. sourcecode:: bash

    $ pip install -r requirements.txt


Check operation
---------------

.. sourcecode:: bash

    $ python gemini.py --version
    0.9.5


Usage
=====

Exec following command to show.

.. sourcecode:: bash

    $ python gemini.py -h

For example

.. sourcecode:: bash

    $ python gemini.py accesslog.txt
