Requirement

Installation
============

First, Clone this repository.


Install requisite packages
--------------------------

.. sourcecode:: bash

    $ pip install -r requirements.txt


Check operation
---------------

.. sourcecode:: bash

    $ python jumeaux.py --version
    0.9.5


Usage
=====

Exec following command to show.

.. sourcecode:: bash

    $ python jumeaux.py -h

For example

.. sourcecode:: bash

    $ python jumeaux.py --config config.yaml accesslog.txt
